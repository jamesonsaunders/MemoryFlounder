## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/jamesonsaunders/MemoryFlounder.git

# Navigate to the project directory
cd memory-flounder

# Install dependencies
npm install
```

### Development

```bash
# Start the development server
npm run dev

# The app will be available at http://localhost:5173
```

### Build

```bash
# Build for production
npm run build:prod

# Preview production build
npm run preview
```

## How to Play

1. Select your preferred game mode (Emoji, Word, or Letter)
2. Choose a grid size based on difficulty
3. Click "Start Game" to begin
4. Click cards to flip them and reveal their content
5. Match pairs by finding two cards with the same content
6. Try to complete the game in as few moves as possible!
